# SHAKE - World Quake Scale 仕様書

作成日：2026年5月1日

---

## 1. アプリ概要

| 項目 | 内容 |
|---|---|
| アプリ名 | SHAKE - World Quake Scale |
| プラットフォーム | iOS / Android（Flutter） |
| カテゴリ | エンターテイメント |
| 価格 | 無料 |
| 収益化 | AdMob（広告） |

### コンセプト
スマホを思い切り振って、加速度を計測。震度（日本）またはMMI（海外）に換算して世界ランキングに登録する。地震の揺れスケールを体感で理解できる教育的エンターテイメントアプリ。

> ※ スマホを振る動作と実際の地震は異なります。あくまで体感目安です。

## 3. 計測仕様

| 項目 | 内容 |
|---|---|
| センサー | 加速度センサー（sensors_plus） |
| 計測時間 | 10秒 |
| スコア算出 | 10秒間のうち上位5秒分の平均値 |
| スコア単位 | m/s²（加速度の生値） |
| ランキング保存 | 加速度の生値で保存 |
| 表示換算 | 日本ユーザー → 震度 / 海外ユーザー → MMI |

### 震度換算テーブル（日本）

| 震度 | 加速度目安（m/s²） |
|---|---|
| 1 | 0.08〜0.25 |
| 2 | 0.25〜0.80 |
| 3 | 0.80〜2.50 |
| 4 | 2.50〜8.00 |
| 5弱 | 8.00〜14.00 |
| 5強 | 14.00〜25.00 |
| 6弱 | 25.00〜40.00 |
| 6強 | 40.00〜60.00 |
| 7 | 60.00以上 |

### MMI換算テーブル（海外）

| MMI | 体感 | 加速度目安（m/s²） |
|---|---|---|
| Ⅰ | 感じない | ほぼ0 |
| Ⅱ〜Ⅲ | ごくわずか | 0.08〜0.25 |
| Ⅳ | 室内で気づく | 0.25〜0.80 |
| Ⅴ | ほぼ全員が感じる | 0.80〜2.50 |
| Ⅵ | 家具が動く | 2.50〜6.30 |
| Ⅶ | 建物に軽微な被害 | 6.30〜12.00 |
| Ⅷ | 建物に被害 | 12.00〜40.00 |
| Ⅸ〜Ⅻ | 甚大な被害 | 40.00以上 |

## 5. UX仕様

### ニックネーム設定
- LOUDと同じ6文字ドラムロール式
- アルファベット大文字

### 体勢整えカウントダウン（10秒）
スタートボタンを押してから計測開始まで10秒。画面は数字を大きく表示。ユーザーは画面を見ないことを前提とした設計。

| タイミング | バイブ強度 |
|---|---|
| 10〜6秒 | 弱 |
| 5〜2秒 | 中 |
| 1秒 | 強 |
| 0秒（計測開始） | 最強 |

### 計測中（10秒）
| 項目 | 内容 |
|---|---|
| バイブ | なし |
| 音 | 無音 |
| 画面 | 計測中表示のみ |

### 計測開始・終了の合図

| タイミング | バイブ | 音 |
|---|---|---|
| 3・2・1カウント | 各カウントで短くバイブ | カウント音あり |
| 計測開始 | 長めバイブ1回 | スタート音あり |
| 計測中 | なし | 無音 |
| 計測終了 | パターン違うバイブ | 終了音あり |

### 結果画面
- 震度（日本）or MMI（海外）
- 加速度の実測値（m/s²）
- 世界順位
- 国別順位
- SNSシェア機能
- インタースティシャル広告（結果画面表示前）

## 6. ランキング構造

| タブ | 内容 | リセット |
|---|---|---|
| 総合（世界） | 歴代最高スコア | なし |
| 総合（国別） | 歴代最高スコア | なし |
| 月間（世界） | 今月のスコア | 毎月リセット |
| 月間（国別） | 今月のスコア | 毎月リセット |
| 殿堂入り | 月間世界1位の歴代アーカイブ | なし |

- ランキングのスコアは加速度の生値（m/s²）で並べる
- 表示時に震度 or MMIに変換する
- 殿堂入りは月間1位のみ、月別アーカイブとしてスクロール表示
